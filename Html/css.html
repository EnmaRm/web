¿Que es CSS?

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es el lenguaje que se usa para dar estilo y diseño a las páginas web. Permite controlar colores, tamaños, posiciones, tipografías, márgenes, bordes, fondos y mucho más. En pocas palabras: HTML estructura, CSS diseña.

¿Como utilizar CSS en HTML?

CSS se puede usar en una página HTML de 3 formas: CSS en línea (inline), CSS interno, CSS externo

CSS en línea

Es cuando el estilo se coloca directamente en una etiqueta HTML, usando el atributo style. style="color: red; font-size:20px;" style="color: red; font-size:20px;">Este texto está en rojo.

CSS Interno

Es cuando los estilos van dentro de la etiqueta style en el mismo archivo HTML, generalmente en head.

Ejemplo:


<head>
    <style>
        h1 {
            color: blue;
            text-align: center;
        }
    </style>
</head>

<body>
    <h1>Título en azul</h1>
</body>

CSS Externo

Es la manera más profesional y recomendada. El CSS externo se coloca en un archivo separado (por ejemplo, styles.css) y se enlaza al documento HTML usando la etiqueta <link>.

En el HTML lo enlazas así:


<!-- Archivo HTML -->
<head>
    <link rel="stylesheet" href="styles.css">
</head>

<body>
    <h1>Título con CSS externo</h1>
    <p>Este texto usa estilos desde un archivo externo.</p>
</body>

<!-- Archivo styles.css -->
h1 {
    color: green;
}

p {
    font-size: 18px;
    color: #333;
}
