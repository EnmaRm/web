¿Que es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad, dinamismo y lógica a las páginas web. Gracias a JavaScript, una página puede reaccionar a acciones del usuario, cambiar contenido, validar formularios, crear animaciones y mucho más.

Mientras HTML se encarga de la estructura y CSS del diseño visual, JavaScript aporta el comportamiento que hace que un sitio web sea dinámico.

¿Como utilizar JavaScript con HTML

JavaScript se puede usar dentro de un documento HTML de tres formas principales: en línea, interno y externo.

1. JavaScript en línea

Se escribe directamente dentro de una etiqueta HTML usando un atributo como onclick.

Ejemplo:

<button onclick="alert('Hola!')">Presióname</button>

2. JavaScript interno

Se escribe dentro de la etiqueta <script> en el mismo archivo HTML.

Ejemplo:


<script>
    function saludar() {
        alert("Hola desde JavaScript interno");
    }
</script>

<button onclick="saludar()">Saludar</button>

3. JavaScript externo

Se coloca en un archivo separado, por ejemplo script.js, y se enlaza al HTML usando la etiqueta <script>.

Ejemplo HTML:


<script src="script.js"></script>

Ejemplo script.js:


alert("JavaScript externo funcionando");
