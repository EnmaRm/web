Atributos globales

Los atributos globales son aquellos que pueden utilizarse en cualquier etiqueta HTML, sin importar el elemento.

1.1 id

Identifica un elemento de forma única dentro de la página. Solo puede haber un mismo id por documento.

1.2 class

Permite agrupar elementos en una misma categoría. Se usa mucho para aplicar estilos con CSS.

1.3 style

Sirve para aplicar estilos directamente en el elemento (CSS en línea).

Atributos form

Estos atributos se usan en la etiqueta <form> para controlar su comportamiento.

2.1 action

Indica la URL o archivo donde se enviarán los datos del formulario.

2.2 method

Define el método de envío de datos: GET o POST.

Atributos input

Los atributos del elemento <input> sirven para definir el tipo de datos, validaciones y comportamiento.

3.1 type

Indica el tipo de entrada, como: text, password, email, number, etc.

3.2 placeholder

Muestra un texto de ejemplo dentro del campo hasta que el usuario escribe.

3.3 value

Asigna un valor inicial al campo input.
