¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es el código fundamental que se usa para estructurar el contenido de una página web. Su función principal es organizar elementos como texto, imágenes, videos y enlaces mediante etiquetas, lo que permite que los navegadores interpreten y muestren la información en un sitio web.

Historia de HTML

HTML fue creado en 1989 por Tim Berners-Lee para compartir información científica. Su primera versión tenía pocas etiquetas y era muy simple. Con el tiempo evolucionó hasta convertirse en un estándar mundial. Organismos como el W3C definieron reglas y mejoras para garantizar compatibilidad. Con HTML5 llegaron videos, audios y nuevas etiquetas semánticas. Hoy es la base de todo sitio web moderno.

Editores

Un editor HTML es un programa usado para escribir y modificar código web. Existen editores simples como el Bloc de Notas y otros avanzados como VS Code. También hay editores visuales WYSIWYG que muestran el resultado mientras editas. Cada editor ofrece herramientas para facilitar la escritura del código. Muchos incluyen autocompletado, resaltado y vista previa. Escoger un buen editor ayuda a trabajar más rápido y con menos errores.

Tu primer Documento HTML

Un documento HTML debe comenzar con la declaración DOCTYPE que define su versión. Luego incluye la etiqueta <html> que encierra todo el contenido. Dentro del <head> se coloca el título y la configuración del documento. En el <body> va todo lo visible, como textos, imágenes y enlaces. Es común iniciar con un título y un párrafo de bienvenida. Este archivo se guarda con extensión .html y se abre en un navegador.

¿Que son Etiquetas en HTML?

Las etiquetas HTML son instrucciones que indican cómo mostrar el contenido. Se escriben entre símbolos < > y suelen tener apertura y cierre. Cada etiqueta tiene una función específica dentro del documento. Algunas sirven para texto, otras para imágenes o para dividir secciones. La correcta estructura de etiquetas garantiza páginas organizadas. Sin etiquetas, un navegador no podría interpretar el contenido de un sitio.

¿Que son los atributos en HTML?

Los atributos añaden información extra a una etiqueta HTML. Se escriben en la etiqueta de apertura usando la forma nombre="valor". Permiten definir características como tamaño, estilo o comportamiento. Ejemplos comunes son src, id, class, href y placeholder. Cada atributo cumple una función según el tipo de etiqueta donde se use. Sin atributos, muchos elementos no tendrían información esencial para mostrarse.

Reglas fundamentales en HTML

HTML requiere una estructura básica para funcionar correctamente. Las etiquetas deben abrirse y cerrarse en orden para evitar errores. Algunas etiquetas como <img> o <br> no necesitan cierre. Es importante mantener buena indentación para leer el código fácilmente. Los atributos siempre van dentro de la etiqueta de apertura. Seguir estas reglas garantiza páginas más claras y compatibles.

Etiquetas para hacer un documento HTML

1. <!DOCTYPE html> Indica que el documento es HTML5. 2. <html> Etiqueta raíz que contiene todo el documento. 3. <head> Incluye información que no se muestra en la página: Título, Metadatos, Enlaces a CSS, Scripts 4. <meta charset="UTF-8"> Permite usar caracteres como ñ, tildes, símbolos, etc. 5. <title> Título que aparece en la pestaña del navegador. 6. <body> Todo el contenido visible de la página: Textos, Imágenes, Formularios, Botones, Tablas, Etc.
